Make sure the last line is: `export default PersoniQAI;` --- ## Step 2 — Deploy to Vercel (3 ways) ### Option A — GitHub (Recommended, easiest) 1. Create a free account at https://github.com 2. Create a new repository called `personiq-ai` 3. Upload all your files to it 4. Go to https://vercel.com → Sign up free 5. Click **"Add New Project"** → Import your GitHub repo 6. Vercel auto-detects Vite → click **Deploy** 7. ✅ Your site is live at `personiq-ai.vercel.app` ### Option B — Vercel CLI (fastest, no GitHub needed) ```bash # Install Node.js from https://nodejs.org first, then: npm install -g vercel cd personiq-ai npm install vercel # Follow the prompts → done! ``` ### Option C — Drag & Drop (no coding tools needed) ```bash # On your computer terminal: npm install npm run build # This creates a "dist/" folder ``` Then go to https://vercel.com/new → drag the `dist/` folder → instant deploy ✅ --- ## Step 3 — Custom Domain (Optional) In your Vercel dashboard: - Go to your project → **Settings → Domains** - Add your domain (e.g. `personiqai.com`) - Follow DNS instructions from your domain registrar --- ## 🛠 Local Development ```bash npm install # install dependencies npm run dev # opens http://localhost:5173 npm run build # creates production build ``` --- ## ✅ Checklist - [ ] `package.json` created - [ ] `vite.config.js` created - [ ] `index.html` created - [ ] `src/main.jsx` created - [ ] `src/index.css` created - [ ] `tailwind.config.js` created - [ ] `postcss.config.js` created - [ ] `src/App.jsx` created (paste full component) - [ ] Pushed to GitHub or ran `vercel` CLI - [ ] Site live on Vercel 🎉
